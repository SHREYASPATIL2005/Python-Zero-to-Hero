# [Python-Zero-to-Hero](https://github.com/SHREYASPATIL2005/Python-Zero-to-Hero)

This workspace contains chapter-wise Python practice files plus two larger projects:

- [Mega_Project_1 - Jarvis]
- [Mega_Project_2 - AI_AutoReply_Bot]

## Folder Guide

- `Chapter 1` to `Chapter 11`: lesson and problem files for core Python topics.
- `Advance_Python_1` and `Advance_Python_2`: intermediate and advanced practice.
- `Mega_Project_1 - Jarvis`: voice assistant project.
- `Mega_Project_2 - AI_AutoReply_Bot`: chat auto-reply project.

## Setup

1. Use the workspace Python environment in `env`.
2. Install packages from `requirements.txt`.
3. Create a local `.env` file in the workspace root if you want to run the Gemini-based projects.

## Environment Variables

- `GEMINI_API_KEY`: required for Gemini API access.
- `GEMINI_MODEL`: optional, defaults to `gemini-2.0-flash`.

## Notes

- Keep `.env` private.
- The project files are intended to stay simple and readable for learning.
